Background

What is MechanicalTask?
MechanicalTask is a toolkit to democratize the launch of projects enabling micro-contributions from web users, using basic knowledge in html/javascript and that you can host on your own website like a youtube video.

What is a micro-contribution?
Micro-contributions are contributions of any kind that someone could quickly make remotely (or not) within a small amount of time. If we think about the diversity of contributions in participatory projects as a long tail with few core people and a large majority of volatile users,the goal of these micro-applications is to tap into the latter segment assuming that the sum of these micro-contributions could leverage a subtantial work.

What kind of micro-contributions?
e.g. analyzing a large flow of individual tweets, classifying a large volume of images, analyzing a large area on a map, segmenting very quickly a long video collaboratively, generating stories via the writing of individual sentences , or any kind of big data processable/generable more easily by volatile participants than the machine.

Features

  • Embed your micro-app like a youtube video

    MechanicalTask is a web service generating micro-applications embeddable on your website like a youtube video or for mobile phones. No need to deploy a server to setup an basic micro-app. Once embedded on a website, users/visitors can start contributing to some tasks. (note: you are responsible to host any required tutorials/doc on your website).

  • store and share easily your data in the cloud

    MechanicalTask is a distributed infrastructure using google fusion tables as database to store your tasks and the answers of volunteers. We use google fusion and not an internal database because it enables any third party to simply view, export, analyze, visualize the raw answers or event insert new tasks in a more transparent way.

  • Copy & remix apps: toward an ecosystem

    You are in charge to code the UI/client-side to collect the data via an HTML/javascript template. But no need to code locally and deploy it on a server, or to install something

    MechanicalTask shortcuts the launch of an app by using the Gist platform as a first place to host or directly edit online your code via their online editor. Give the URL of your Gist snippet of code and MechanicalTask will generate or update the app on the fly when changes are detected.

    By using Gist , we integrate its social coding features in the building of micro-apps e.g. (public code, easy remix/fork, code comments) and facilitate the emergence of eco-system with best practices in micro-tasking. For instance newcomers can easily fork/remix existing micro-apps in few minutes.
