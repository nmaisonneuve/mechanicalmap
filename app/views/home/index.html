What is microapp.io?

MicroApp.io is a Cloud Toolkit democratizing the building of participatory apps that leverage micro-contributions from web / mobile users for lightweight programmers by cloning apps and coding online . Zero installation is required (no server+database to deploy) thanks to Cloud infrastructures . Embed your app on your own web site like a youtube video. Such infrastructure and its open policy (open app = open data + open code) will facilitate the emergence of an eco-system of micro-apps with best practices.

What is a micro-contribution?

We live in a period of continous partial attention . Micro-contributions are contributions of any kind that someone could quickly make remotely (or not) within a small amount of time. If we think about the diversity of contributions in participatory projects as a long tail with few core people and a large majority of volatile users,the goal of these micro-applications is to tap into the latter segment assuming that the sum of these micro-contributions could leverage a subtantial work (but different from what hard-core contributors will produce). Up to you to request repetitive or not, fun or boring tasks.

Make your crowdsourcing project

  • Minimal setup

    No server + database to install, no deployment process, no code editor to install. MicroApp has a green/minimalist approach by using web eco-system i.e. existing (cloud) services . If you want a community+forum for your app, create a Facebook page or a Google group

  • Open App = Open Data + Open Code

    MicroApp has an open policy: You data and your code are at any time accessible by anyone (read-only for non-admin) thanks to Cloud infrastructure. This policy will facilitate the emergence of eco-system with best practices in micro-tasking.

  • Cloud coding

    Code your app anywhere, anytime by directly using an integrated web code editor and update your app on the fly (+ management of your code versioning via Github ). This shortcuts the redeployment / protoptyping process .

  • Clone apps

    Anyone can clone existing apps and modify the data or the code to generate their own variant crowdsourcing projects in few minutes. Experiment easily different UI / task design by launching several cloned apps.

  • Workflow design

    by default we use a parallel worfklow with redundancy mechanism. but you are free to design your own workflow e.g.iterative model, or inject the answer of a micro-app as input for other apps creating interaction between apps (TODO add section: "From Solver to Designer" about crowdsourcing/self-organisation the decomposition/design of tasks to solve a pb)

Leverage participation

  • Embeddable and spreadable over the web

    Embed any generated app on your web site like a Youtube video. Once embedded, web visitors can directly play with the app, or also embed the app to their websites to support your action beyond traditional advertising.

  • Mobile Compatible

    Code an app directly from your mobile thanks to the online editor. We also generated the associated QRCode stick to access to app in full screen mode, so you can print it to advertise your app in the physical world.

  • Mechanical Turk compatible

    You can use the Amazon mechanical turk service as fast and chip simulator to test your app or get tasks done by turkers. You need to inject a small code into your app
