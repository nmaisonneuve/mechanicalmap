What is microapp.io?

MicroApp.io is a toolkit to help lightweight web programmers make crowdsourcing apps requesting micro-contributions from web / mobile users. Zero installation is required thanks existing cloud infrastructure.

Create a crowdsourcing app in few minutes: clone existing apps (data+code) , edit online the code or the data online and embed your app on your own web sitelike a youtube video. We have an open policy: open app = open data + open code to facilitate the emergence of an eco-system of micro-apps with best practices.

What is a micro-contribution?

We live in a period of continous partial attention. Micro-contributions are contributions of any kind that someone could quickly make within a small amount of time.

The contributions in a participatory project follows generally a long tail: few core people and a large majority of volatile users contributing few times.

The goal of these micro-apps is to tap into the latter segment assuming that the sum of these micro-contributions could leverage a subtantial work (but different from what hard-core contributors will produce). Up to you to request diverse & fun micro-tasks or repetitive & boring ones.

Features to make your crowdsourcing project

  • Green / Minimal setup

    No server + database to install, no deployment process, no code editor to install. MicroApp has a green/minimalist approach by reusing existing (cloud) services . If you want a community+forum for your app, create a Facebook page or a Google group

  • Open App = Open Data + Open Code

    MicroApp has an open policy: You data and your code are at any time accessible by anyone (read-only for non-admin) thanks to Cloud infrastructure. This policy will facilitate the emergence of eco-system with best practices in micro-tasking.

  • Cloud data

    We use google fusion tables to store your data and not an internal database because anyone can easily view, export, analyze, visualize the raw data. .

  • Cloud coding

    Code your app anywhere, anytime by directly using an integrated web code editor and update your app on the fly (+ management of your code versioning via Github ). This shortcuts the redeployment / protoptyping process .

  • Clone apps

    Anyone can clone existing apps and modify the data or the code to generate their own variant crowdsourcing projects in few minutes. Experiment easily different UI / task design by launching several cloned apps.

  • Workflow design

    Design your own dynamic workflow at the user level e.g. intro -> assessment/exercice, -> real task, and at the collective level e.g. iterative or parallel collective problem solving model, or inject the answer of a micro-app as input for other apps, creating interaction between apps

Features to leverage participation

  • Embeddable app like a youtube video

    Embed any generated app on your web site like a Youtube video. Once embedded, web visitors can directly play with the app, or also embed the app to their websites to support your action beyond traditional advertising.

  • Mobile Compatible

    Code an app directly from your mobile thanks to the online editor. We also generated the associated QRCode stick to access to app in full screen mode, so you can print it to advertise your app in the physical world.

  • Mechanical Turk compatible

    You can use the Amazon mechanical turk service as fast and chip simulator to test your app or get tasks done by turkers. You need to inject a small code into your app

  • Captcha with a purpose

    Since any app is embeddable, you can easily create captcha with a purpose for any website supporting your action.
