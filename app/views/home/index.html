Introducing microapp.io

What is microapp.io?

MicroApp.io is a toolkit to help lightweight Javascript/HTML5 developers building crowdsourcing apps requesting micro-contributions.

Create a crowdsourcing app in few minutes: clone existing apps (data+code), edit online the code or the data and embed your app on your own web site like a youtube video. Zero installation is required thanks to existing cloud infrastructures.

What is a micro-contribution?

We live in a period of (continous) partial attention. Micro-contributions are contributions of any kind that someone could quickly make within a small amount of time.

In a participatory project contributions follow generally a long tail: few core people and a large majority of volatile users contributing few times. The goal of microapp.io is to tap into the latter segment assuming that the sum of these micro-contributions could leverage a subtantial work (but different from what hard-core contributors will produce). Up to you to request diverse & fun contributions or repetitive & boring ones.

Make your crowdsourcing project

minimal setup

No server + database to install, no deployment process. Microapp.io is green and reuses existing web (cloud) services. If you want a community forum, create a Facebook page or a Google group.

open code + open data

Microapp.io has an open policy: open app = open source code + open data. That means your data and your code are at any time accessible by anyone (in read-only mode). Such policy facilitates the emergence of best practices

Clone apps

Anyone can clone existing apps and modify the data or the code to launch their own variant crowdsourcing project in few minutes, or prototype and test different versions

Javascript is enough

no need of server-side code (python, ruby or php). Only some knowledge in HTML(5)/ Javascript is enough. Microapp provides a Javascript framework based on Backbone.js to build microapps.

Cloud data

We use google fusion tables to store your data and not an internal database because anyone can easily view, export, analyze, visualize the raw data.

Cloud coding

Code your app anywhere, anytime by directly using a online code editor to update your app on the fly and thus to shortcut prototyping process (versioning management of your code via Github).

Workflow design

Design your own dynamic workflow at the user level (e.g. intro, tutorial,test, real task) and at the collective level e.g. iterative or parallel collective problem solving models, or pipeline of apps (output of an app = input of another)

Leverage participation

Embedable/spreadable app

Embed your microapp directly on your own web site like a Youtube video. Once embedded, your visitors can play with the app, or embed the app on their websites to support your action beyond traditional advertising.

Mobile Compatible

Code an app directly from your mobile thanks to the online editor. We also generated the associated QRCode stick to access to app in full screen mode, so you can print it to advertise your app in the physical world.

Captcha with a purpose

Since any app is embeddable, you can easily create captcha with a purpose for any website supporting your action.

Mechanical Turk compatible

Microapp provides a layer to make your app compatible with Amazon mechanical turk service (AMT) in a transparent way. Use AMT as fast and chip 'human simulator' to test your app or get directly some tasks done.
