Background

What is volatileTask?
volatileTask is a Cloud Toolkit to democratize the launch of projects enabling micro-contributions from web users, using basic knowledge in html/javascript and that you can host on your own website like a youtube video. Zero installation or deployment of a server is required: You data +your code are stored/shared and clonable thanks to cloud infrastructures

What is a micro-contribution?
Micro-contributions are contributions of any kind that someone could quickly make remotely (or not) within a small amount of time. If we think about the diversity of contributions in participatory projects as a long tail with few core people and a large majority of volatile users,the goal of these micro-applications is to tap into the latter segment assuming that the sum of these micro-contributions could leverage a subtantial work.

What kind of micro-contributions?
e.g. analyzing a large flow of individual tweets, classifying a large volume of images, analyzing a large area on a map, segmenting very quickly a long video collaboratively, generating stories via the writing of individual sentences , or any kind of big data processable/generable more easily by volatile participants than the machine.

Features

  • Spreadable over the web

    volatileTask generates micro-apps embeddable on your own website like a youtube video and usable for mobile phones. Once embedded, web visitors can directly complete some tasks, or also embed the app to their websites to support your action beyond traditional advertising.
    Note you are responsible to host any required tutorials/doc on your website. We do not host community & forums etc

  • Cloudable = zero installation + tranparency

    No need to deploy a server to setup a crowdsourcing project.
    Data: volatileTask is a distributed infrastructure using google fusion tables as database to store your tasks and the answers of volunteers. We use google fusion and not an internal database because it enables any third party to simply view, export, analyze, visualize the raw answers or event insert new tasks in a more transparent way.
    Code: You are in charge to code the UI to collect the data via an HTML/javascript template. volatileTask enables you to direclty edit your code online, You app is updated on the fly, shortcuting the traditional redeployment cycle. .Furthermore you code is saved into Gist platform to manage versioning .

  • Clonable

    Anyone has access to the data and the code of any micro-app. They can slo clone existing micro-apps to generate variant ones in few minutes. Such feature facilitates the emergence of eco-system with best practices in micro-tasking.
