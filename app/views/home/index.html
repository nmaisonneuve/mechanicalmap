Mechanical Mapping
Platform for experimentating crowdmapping projects using a micro-task driven approach.

Create your project Contribute to a project

No need to deploy a server or a database to create your own micro-task platform.
Build your project by creating or remixing existing micro-Task template code (html/javascript) and embed it on your own web page.

Background

This cloud toolkit is an attempt to democratize the development and experimentation of microtasking projects to anyone having some basic knowledge about html/javascript. It is based on two main principles.

  • Zero server

    We live in the age of cloud computing so no need to set up your own server and a database. Our goal is also *not* to host projects. Actually we are like a search engine maintaining an index of tasks for volunteers. We index your tasks via a google fusion table (input). We manage the allocation of tasks and send the results to another fusion table (output) we generated but on which you have the ownership.

    You are only in charge of the user interface to collect the data via a HTML/javascript template. Some helpers/libraries are available. From this script, we generate a web widget you can embed to your web page or share like a youtube video (or used it as banner on your partners' web sites instead of advertising).

  • Openess: toward a micro-tasking ecosystem

    We think that what is valuable is not only the data but also the mechanism you used to collect them. So if you generate a project via this platform your html/script template will be automatically shared with the community so other people can transform or combine it. We think such openess will facilitate the emergence of best practices in micro tasking, and leverage more quickly a potential community.
